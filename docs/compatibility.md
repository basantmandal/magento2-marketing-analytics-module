# Compatibility

## Magento Versions

- Magento Open Source 2.4.x
- Adobe Commerce 2.4.x

## PHP Versions

- PHP 8.1
- PHP 8.2
- PHP 8.3
- PHP 8.4

## Dependencies

| Package | Version |
|---------|---------|
| `php` | ^8.1 \|\| ^8.2 \|\| ^8.3 \|\| ^8.4 |
| `magento/framework` | ^103.0.0 |
| `hk2/core` | ^1.0 |

## Module Dependencies

This module requires the following Magento modules to be enabled:

- `Magento_Backend`
- `Magento_Config`
- `Magento_Store`
- `Magento_Cron`
- `HK2_Core`

## Browser Compatibility

The frontend tracking scripts (Google Analytics and Facebook Pixel) are compatible with all modern browsers including:

- Google Chrome
- Mozilla Firefox
- Apple Safari
- Microsoft Edge
- Opera

## Third-Party Services

- **Google Analytics / GA4** — Requires a valid Google Analytics 4 Measurement ID
- **Facebook (Meta) Pixel** — Requires a valid Facebook Pixel ID
- **Google Merchant Center** — Required for product feed integration

## Editions

This module supports both Magento Open Source and Adobe Commerce editions.
